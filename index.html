Ode Programming Language

Ode is a stack-based, concatenative, functional programming language created by myself, with an interactive online development environment written in JavaScript.Â¨

Ode is inspired primarily by the Joy programming language, which Ode aims to implement fully.

Use Ode

Ode uses features of HTML 5, which means it will only work on newer browsers (as of Spring 2011). I have tested the current version of Ode on the following browsers:

  • Firefox 4
  • Chrome 10
  • Internet Explorer 9
